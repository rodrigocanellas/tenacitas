class tnct\+::lib\+::alg\+::internal\+::log\+\_\+wrapper$<$ use $>$}

Thread safe writing of log messages to console (cerr/cout/clog) A log message has the format\+: log-\/level$\vert$timestamp$\vert$log\+\_\+file-\/name$\vert$thread-\/id$\vert$line-\/number$\vert$contents. 

{\ttfamily log-\/level} is TNCT\+\_\+\+LOG\+\_\+\+TRA, TNCT\+\_\+\+LOG\+\_\+\+DEB, TNCT\+\_\+\+LOG\+\_\+\+INF, TNCT\+\_\+\+LOG\+\_\+\+WAR, TNCT\+\_\+\+LOG\+\_\+\+ERR or TNCT\+\_\+\+LOG\+\_\+\+FAT; {\ttfamily timestamp} is a EPOCH number in microsecs; {\ttfamily log\+\_\+file-\/name} is the name of the \mbox{\hyperlink{structtnct_1_1lib_1_1alg_1_1internal_1_1log__file}{log\+\_\+file}} where the message is being writen; {\ttfamily thread-\/id} is the id of the thread that is logging; {\ttfamily line-\/number} is the number of the line that is logging; contents are the log message specific contents
