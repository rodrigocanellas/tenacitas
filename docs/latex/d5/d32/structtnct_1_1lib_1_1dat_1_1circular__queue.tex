struct tnct\+::lib\+::dat\+::circular\+\_\+queue$<$ t\+\_\+data $>$}

Implements a circular queue which size is increased if it becomes full. 

The purpose is to avoid unnecessary memory allocations to create nodes in the queue by reusing nodes which data have been read
