Detailed Description

template<bool use = true>
class tnct::lib::alg::internal::log_wrapper< use >

Thread safe writing of log messages to console (cerr/cout/clog) A log message has the format: log-level|timestamp|log_file-name|thread-id|line-number|contents.

log-level is TNCT_LOG_TRA, TNCT_LOG_DEB, TNCT_LOG_INF, TNCT_LOG_WAR, TNCT_LOG_ERR or TNCT_LOG_FAT; timestamp is a EPOCH number in microsecs; log_file-name is the name of the log_file where the message is being writen; thread-id is the id of the thread that is logging; line-number is the number of the line that is logging; contents are the log message specific contents
