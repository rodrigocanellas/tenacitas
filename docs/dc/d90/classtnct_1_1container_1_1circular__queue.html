Detailed Description

template<log::cpt::logger t_logger, typename t_data, std::size_t t_initial_size, std::size_t t_incremental_size = t_initial_size / 2>
class tnct::container::circular_queue< t_logger, t_data, t_initial_size, t_incremental_size >

Implements a circular queue which size is increased if it becomes full.

The purpose is to avoid unnecessary memory allocations to create nodes in the queue by reusing nodes which data have been read

Template Parameters
t_datadefines the types of the data contained in the queue
