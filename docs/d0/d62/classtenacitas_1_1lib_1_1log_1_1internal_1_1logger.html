Detailed Description

template<traits::log_writer t_log_writer>
class tenacitas::lib::log::internal::logger< t_log_writer >

Thread safe writing of log messages.

A log message has the format: log-level|timestamp|thread-id|function-name|line-number|contents

log-level is TST, TRA, DEB, INF, WAR, ERR or FAT; timestamp is a EPOCH number in microsecs; thread-id is the id of the thread that is logging; funcition-name is the name of the function line-number is the number of the line that is logging; contents are the log message specific contents
