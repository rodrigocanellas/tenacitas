Detailed Description

Thread safe writing of log messages A log message has the format: log-level|timestamp|file-name|thread-id|line-number|contents.

log-level is TRA, DEB, INF, WAR, ERR or FAT; timestamp is a EPOCH number in microsecs; file-name is the name of the file where the message is being writen; thread-id is the id of the thread that is logging; line-number is the number of the line that is logging; contents are the log message specific contents
