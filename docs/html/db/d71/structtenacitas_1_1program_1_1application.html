Detailed Description

Entry point for an application.

This class allows asynchronously execution of a function, which must use tenacitas::async::dispatch<tenacitas::program::exit_app>() function to send a tenacitas::program::exit_app to end the application

Examples
application_000/main.cpp, and reader_000/main.cpp.
