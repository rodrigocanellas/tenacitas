Signature of the function that will be called in each round of the loop.

Parameters
p_boolis a means for the object that will call the handler function to inform that the handler may stop working, as the timeout defined for it to finish its work has expired. So, the handler function should, as most as possible, between its instructions, to check for the value o p_bool, in order to continue to the following instruction, or to interrupt its execution
