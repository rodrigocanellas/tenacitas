template<typename t_event >
using tenacitas::async::handler_t = typedef std::function<void(ptr<bool> p_bool, t_event &&p_event)>

Type of function that handles an event.

Template Parameters
t_eventis the type of event that will be handled It must be default constructible, copy constructible, copy assignable, move constructible, move assignable and must have the output operator (<<) defined
Parameters
p_boolis a means for the object that will call the handler function to inform that the handler may stop working, as the timeout defined for it to finish its work has expired. So, the handler function should, as most as possible, between its instructions, to check for the value o p_bool, in order to continue to the following instruction, or to interrupt its execution
p_eventis the event to be handled
