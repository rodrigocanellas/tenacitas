Detailed Description

Synchronous execution, but with timeout control.

Template Parameters
t_timetype of the time used to specify timeout
t_functionfunction to be executed. It must have this signature: std::function<void(tenacitas::type::ptr<bool> p_bool, t_params... p_params)>. The p_bool is how execute will let the function know that it has exceed the time limit for its execution. The function should between its instructions to check for p_bool, to check if it should stop executing.
t_params...type of the parameters to the function
Parameters
p_timeoutmaximum amount of time that the function has to finish its execution. It must be one of the types defined in std::chrono
p_functionfunction to execute in p_timeout time, with the signature described above.
p_param...arguments to p_function
Returns
true if p_function executed in less than p_timeout, or false otherwise.
