Detailed Description

template<typename t_ret>
struct tenacitas::async::internal::executer_t< t_ret >

Synchronous execution, but with timeout control.

Template Parameters
t_rettype that the function to be executed returns
t_timetype of the time used to specify timeout
t_functionfunction to be executed. It must have this signature: std::function<t_ret(tenacitas::type::ptr<bool> p_bool, t_params... p_params)>. The p_bool is how execute will let the function know that it has exceed the time limit for its execution. The function should between its instructions to check for p_bool, to check if it should stop executing.
t_params...type of the parameters to the function
Parameters
p_timeoutmaximum amount of time that the function has to finish its execution. It must be one of the types defined in std::chrono
p_functionfunction to execute in p_timeout time, with the signature described above.
p_param...arguments to p_function
Returns
if p_function executed in less than p_timeout, a valid t_ret value; or an empty std::optional, otherwise.
