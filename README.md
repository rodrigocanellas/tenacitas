# tenacitas


![tenacitas](doc/tenacitas_logo.png)

**tenacitas** is a latin word which means tenacity. 


# This repository hosts these C++ projects: #

    calendar - classes to support time manipulation and conversion.

    concurrent - Sleeping loop, thread pool and publisher/subscriber are implemented in this project.

    logger - Thread-safe logs for std::cout, std::cerr, std::clog. 
             Also class to log in files, which allows definition of retention.
		 
    number - Classes with a variety of number manipulations, such as formating.

    program - Classes related to programs, such as program options.

    tester - Simple class to help creating uint tests.
