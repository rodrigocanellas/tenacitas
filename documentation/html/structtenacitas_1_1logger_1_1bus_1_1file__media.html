Detailed Description

template<typename t_controller>
struct tenacitas::logger::bus::file_media< t_controller >

The file_media struct implements a t_media type to be used in a tenacitas::logger::bus::log class, allowing log messages to be written to a file.

Template Parameters
t_controlleris a class that will help controlling how the messages will be written, how the files will be created, among other things. It must implement:
  • move constructor
  • move assignment
  • void remove()
    which will control when a file must be removed
  • void rename(const std::string &)
    which will rename a file when no more log messages can be written to the file
  • bool renew(uint32_t)
    which will return true if the file must be renewed, i.e., if no more messages shall be written to the file, and a new one must be created. To help making the decision, the file size will be passed to the method, in a uint32_t value
  • std::string name()
    which will provide a name for the file
Attention
There is a default controller created, in file file_controller.h
