Detailed Description

template<typename t_data>
class tenacitas::concurrent::business::thread_pool< t_data >

thread_pool implements a thread pool, which allows to use the producer/consumer pattern

Template Parameters
t_datais the type of data inserted into the pool, so that registered work functions can "fight" among each other to get the data to process it. t_data must be default constructable, moveable and copiable
Parameters
t_datais the type of the data to be handled. If it is not void, it must be:
  • default constructible
  • move constructible
