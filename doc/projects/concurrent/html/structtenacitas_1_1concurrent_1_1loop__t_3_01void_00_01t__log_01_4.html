Detailed Description

template<typename t_log>
struct tenacitas::concurrent::loop_t< void, t_log >

loop implements a generic loop

A loop needs a work function, that will execute a defined work at each round of the loop; an amount of time that the work function has to execute; and a break function, that indicates when the loop should break

In this specialization of the loop class, the work function actually does needs a data, so the type parameter is set to void.
