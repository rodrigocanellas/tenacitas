template<typename t_msg, typename t_log, typename t_time = std::chrono::milliseconds>
class tenacitas::concurrent::dispatcher_t< t_msg, t_log, t_time >

dispatcher implements a publisher/subscriber pattern

Parameters
t_datais the type of the data to be handled. If it is not void, it must be:
  • default constructible
  • move constructible
Template Parameters
